### 여행 코스 계획

- 여행지 검색을 통해 지도에서 위치를 확인하고, 장소를 추가하면 우측의 계획표에 등록
- 드래그 앤 드롭으로 위치를 변경하고 변경된 위치에 맞게 지도 순서 변경
- 각 일차별로 간단한 메모 작성 가능

![plan-ezgif com-optimize](https://github.com/user-attachments/assets/66f2ae75-2721-406f-8065-36bdc5697a80)
